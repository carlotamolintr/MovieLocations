      margin:0 auto; 0 es para arriba-abajo y auto para izquierda-derecha. Significa que el margen izquierdo y derecho tomará el margen automático de acuerdo con el ancho del elemento y el ancho del contenedor.

      ¿Qué es el margin top CSS?
      La propiedad margin-top establece el margen superior de un elemento especificando una longitud o un porcentaje. Los valores en porcentaje se refieren al ancho del elemento padre.

      ¿Cómo hacer un margen en CSS?
      Si queremos modificar los márgenes de una página web deberemos de utilizar cuatro propiedades CSS:
      1. margin-bottom, para el margen inferior.
      2. margin-left, para el margen izquierdo.
      3. margin-right, para el margen derecho.
      4. margin-top, para el margen superior.
