      margin:0 auto; 0 es para arriba-abajo y auto para izquierda-derecha. Significa que el margen izquierdo y derecho tomar el margen automtico de acuerdo con el ancho del elemento y el ancho del contenedor.

      Cmo hacer un margen en CSS?
      Si queremos modificar los mrgenes de una pgina web deberemos de utilizar cuatro propiedades CSS:
      1. margin-bottom, para el margen inferior.
      2. margin-left, para el margen izquierdo.
      3. margin-right, para el margen derecho.
      4. margin-top, para el margen superior.
